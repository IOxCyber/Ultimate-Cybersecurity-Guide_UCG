## If you find a suspicious process in Advanced Hunting Tab in Hunting Section of EDR:

1. Check legitimacy: Verify file path, digital signature, and hash (via VirusTotal).

2. Analyze behavior: Look for unusual network calls, child processes, persistence, or privilege escalation.

3. Correlate logs: See related alerts (EDR/SIEM) and user actions.

4. Isolate & contain: If confirmed malicious, quarantine host or block IOCs.

5. Document & report: Note findings, indicators, and remediation steps.
