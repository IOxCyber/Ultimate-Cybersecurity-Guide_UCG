# How to respond to INC & Events


### 1. Connect to Linux Assets:
- EDR > Assets(only Linux) > select `Connect to Host` > run cmds eg. help > End session

### 2. Forensics For Windows:
- EDR > Forensics > Request Forensic Data > Choose Win asset from menu > select Data Collection Type

### Data collection type:
1. System Forensics::
- Log data of User Accounts, files & processei, event logs, network logs & persistence related data.

2. Process Dump:
- Log data of memory of running processes on the endpoints.
