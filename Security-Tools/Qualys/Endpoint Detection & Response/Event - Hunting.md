1. Access Events from EDR > HUNTING > Events
2. Can be filtered by Objects like `Process, File, Registry, Network`
3. Event Actions includes, Running, terminated, deleted, created, rename etc
4. Severity Score (Range from 0 to 9)
5. Remediation Action: Kill Process
6. Shows Activities in Event details (MITRE ATT&CK Technique/Ways, Tactics/What)
7. Image: Shows full PATH, Hash Value, File size, references (virus Total)
8. Process Tree (all processes including spawned)
9. Exempted Events: List of all Removed Events
10. Advance Hunting: QQL on collected raw events to find any suspicious activity.



> `No Action Required` means there is no malware activity for the event.
