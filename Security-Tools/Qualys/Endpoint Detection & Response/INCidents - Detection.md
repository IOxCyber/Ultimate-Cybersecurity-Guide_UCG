# Detection (Incident): To analyse an INC

1. EDR > Detection > List of INCidents 
2. Can be filtered by Malware Category, Behaviour Detection, Incident Status, Assignee

> Incident Severity score comes from Highest severity score.

3. Check INC details, Summary, Process tree(Relationship b/w processes & files), Comments, Activity Logs(any action taken), Risk & Exploits(Any misconfiguration/Vuln)

> Risk & Exploits tab allows deploy mitigations & Patches directly from INC details.

4. Timeline tab: Via Action option, a user can change the INC status, assign INC to user, add a comment or perform investigative actions.
