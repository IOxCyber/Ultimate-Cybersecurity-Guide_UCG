## Snyk:
- Open Source Version available, SAST/SCA tool.
- Best for Py, JS languages.
- Can be Integrated with IDE CI/CD.
- Code can be uploaded directly via GitHub, CLI or Bit bucket etc.

### SAST: Static AppSec Testing::
- Scan for vulnerabilities at code level without executing the code.
### SCA: Software Composition Analysis::
- Scan the dependencies & 3rd party libs for vulns.
